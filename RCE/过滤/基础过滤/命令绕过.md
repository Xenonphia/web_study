# file -f
File 用以识别文件类型

-f 指定一个文本文件，该文件的**每一行**对应一个要检测的文件路径（绝对路径 / 相对路径均可）

因此file -f flag.php意味着会检测php文件中每一行内容，因此会造成报错并且回显出原始内容
因为文件内容并不是一个真实的有效的路径

![](assets/命令绕过/file-20260109120709251%201.png)

# grep
查找指定<mark style="background: #FFB8EBA6;">字符串</mark>
Grep fl fl*
第一个fl是匹配模式，匹配文件中含有fl的字符串
第二个为fl*通配符搜索fl开头的所有文件

![](assets/命令绕过/file-20260109120709251.png)

因此grep fl fl*就是在fl*的文件里面寻找有fl的内容

*可选参数

![](assets/命令绕过/file-20260109120709250%201.png)

![](assets/命令绕过/file-20260109120709250.png)

此时，又可以结合之前的管道符|将前面的命令作为后面的参数
例如cat f* | grep flag
等价于 grep flag f*
