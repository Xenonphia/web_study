# base64
Cat flag.php -> Y2F0IGZsYWcucGhw

![](assets/编码绕过/file-20260109120704638.png)

意味着可以通过编码后解码得到需要的命令和参数
因此只需要解码后再执行
Zsh环境下，在最后可以再加入参数 | sh
让其变成命令去执行
以及bash等

![](assets/编码绕过/file-20260109120704637.png)

在linux也可以使用反引号以及$()的形式`变成命令去执行

![](assets/编码绕过/file-20260109120704636%201.png)

![](assets/编码绕过/file-20260109120704636.png)

# hex
ASCII码
例如tac flag.php -> 74616320666c61672e706870
(经过hex编码)

![](assets/编码绕过/file-20260109120704635.png)

在Linux下执行 使用xxd查看
其中xxd –r -p的作用就是将hex编码还原

![](assets/编码绕过/file-20260109120704634%201.png)

-r -p将纯十六进制转储的反向输出打印为了ASCII格式
