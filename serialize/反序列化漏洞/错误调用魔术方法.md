# \_\_call
当调用了一个不存在的方法的时候触发
参数：2个参数传参\$arg1,\$arg2
返回值：调用不存在的方法名称和参数

```php
<?php

//highlight_file(__FILE__);

error_reporting(0);

class User {

    public function __call($arg1,$arg2)

    {

        echo "$arg1,$arg2[0]";

          }

}

$test = new User() ;

$test -> guc('a');

?>
```

调用不存在的函数guc()和参数a分别按照格式回显

# \_\_callStatic
静态调用或者调用成员常量时使用的方法不存在时触发
参数和返回值同\_\_call
